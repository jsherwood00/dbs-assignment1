# Maison Vertige — Rapid Prototyping

A fictional luxury fragrance house. 
"Five fragrances. Forty years apart."
Founded ~1984. One address (Paris). Five numbered fragrances. 
Quiet, restrained, old-money sensibility. No logos, no ambassadors, no explanations.

## Rules
- Pure HTML and CSS only. No JS, no frameworks, no external APIs.
- Each version is a standalone file: v1.html, v2.html, etc.
- index.html is the gallery page linking all versions.
